Talabat-Lite

Talabat-Lite is a server-client application for android. It facilitates food, goods or medicines delivery and other services, connecting customers with local vendors and managing delivery personnel.
The server is written in C++ communicating with client over HTTP using crow (microframework for the web, uses routing similar to Python's Flask). For database it uses SQLite C/C++ interface.

ChessOverSockets

Simple 2 player Chess console game connecting over sockets written in C++.
Applying OOP concepts like Polymorphism and using Dynamic Memory Allocation.
The game is offering 2 options, Search for players and Invite a player.
Invites a player using his IP address and port number or Listens to coming invites using the socket binding to a network interface.
Detects win, lose, checkmate, chess rules, and most of draw cases.
